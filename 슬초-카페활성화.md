## 로드맵

### 1단계 — 카페 활성화 자동화 (ASAP)
- 목표: 컨텐츠가 자주 올라오는 공간 만들기
- 방법: 초등맘 카페(20만) 인기 게시글 → 변형·심화·확장 → 슬초 스타일로 재작성 → 자동 업로드
- 대상 채널: 네이버 카페, 네이버 블로그

### 2단계 — 문해력 컨텐츠 추가 (2026년 7월~)
- 전제: 펜타스쿨 커리큘럼 확정 이후
- 방법: 정보형 포스팅 + 카드뉴스 형태
- 목적: "문해력 자료가 많은 공간" 이미지 구축, 팔로워 확대

### 3단계 — 펜타스쿨 연구수업 모집 (2026년 12월)
- 2단계 누적된 팔로워 대상으로 연구수업 신청 모집 공개

## 기술 구현 계획

### 콘텐츠 수집
- 네이버 검색 API — 초등맘 카페 인기 키워드 검색 (공식, 무료 25,000 호출/일)
- (선택) Playwright 크롤링 — 본문 직접 수집

### 콘텐츠 생성
- Claude API — 수집된 주제를 슬초 스타일로 변형·심화

### 자동 업로드
- 네이버 블로그 API
- 네이버 카페 API (카페 운영자 권한 필요)
- 카카오 비즈채널 API
- Meta Graph API (인스타그램)

### 참고
- 기존 리마크에듀 블로그 자동 포스팅 시스템 구조 재활용 가능
- 네이버 디벨로퍼스 연동 안건 선처리 필요
