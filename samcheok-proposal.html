01 · BACKGROUND

왜, 자기주도 캠프인가.

초등 고학년은 학습 습관의 방향이 결정되는 분기점입니다. 이 시기에 형성된 스스로 공부하는 힘은 중·고등 학업 성취의 결정적 토대가 됩니다.

01
사교육 의존의 한계
지시 받는 공부만 해 온 학생은 중학교 진학 후 급증하는 학습량 앞에서 방향을 잃기 쉽습니다. '시키는 공부'는 오래 지속되지 않습니다.
02
지역 교육 격차
수도권에 비해 지방 학생들은 다양한 학습 롤모델을 접할 기회가 제한적입니다. 또래와 멘토를 통한 자극이 절실한 상황입니다.
03
진단 없는 일방 처방
학생마다 학습 유형과 자기주도 수준이 다른데, 획일적 커리큘럼으로는 개인의 문제 지점에 접근할 수 없습니다.
04
가정 연계 부재
학원과 가정이 따로 돌아갈 때 학습 효과는 쉽게 흩어집니다. 가정이 '관리자'가 아닌 '환경 설계자'로 전환되어야 합니다.

본 프로그램의 목적

삼척 초등 5·6학년 학생이 자신의 학습 유형과 실행 수준을 이해하고,
서울대 멘토의 밀착 코칭으로 스스로 계획하고 실천하는 습관을 4주 안에 구조화한다.

4대 차별화 포인트

①
서울대 재학생 4인 전담 멘토진
최상위 자기주도 학습을 구현해 낸 대학생들이 초등 학생과 4주간 1:5로 매칭됩니다. 관계 기반 멘토링으로 '지식'이 아닌 '습관'을 전수합니다.
②
이중 진단 기반 맞춤 설계
학습 유형 진단과 자기주도력 진단 — 두 축의 데이터로 학생별 약점을 특정하고 개인 최적 코칭 전략을 설계합니다. 획일적 커리큘럼이 놓치는 개인 병목에 접근합니다.
③
주중 매일 저녁 9시 플래너 코칭
대면 수업 1회로 끝나지 않습니다. 전담 코치 3명이 매일 밤 9시 플래너에 개별 코멘트를 남기며, 행동 수정 중심 피드백으로 4주 내내 실행을 관리합니다.
④
가정 연계 리포트 & 실행 가이드
캠프 종료 시 학생·학부모에게 진단 결과와 실천 전략 리포트를 제공합니다. 학부모를 '환경 설계자'로 전환시켜 4주 이후에도 습관이 유지되게 합니다.

가정에는 자녀 맞춤 학습 가이드가 남습니다. 습관은 캠프가 끝나도 계속됩니다.

09 · DIAGNOSTIC TOOLS

두 축의 진단으로 정확한 출발점을.

'학습 유형'과 '자기주도력' 두 진단을 통해 학생의 현재 상태를 다각도로 파악하고, 프로그램 내·외에서 지속적으로 활용합니다.

TOOL 01
학습 유형 진단

학생의 학습 습관·선호 방식·정보 처리 유형을 분석하여 맞춤형 공부법을 설계합니다.

프로그램 내 즉시 적용

진단 결과를 바탕으로 개인별 학습 특성 분석 → 맞춤형 코칭 전략에 반영. 주간 플래너 작성·피드백에 지속 적용.

개인 맞춤 리포트

학생·학부모에게 학습 유형·행동 패턴·강점·추천 학습 전략을 담은 분석 리포트 제공.

프로그램 이후 활용

학생은 자신의 유형을 기준으로 지속 학습 계획 수립, 학부모는 통제 중심이 아닌 지원 중심 개입으로 전환.

TOOL 02
자기주도력 진단

학생의 시작·지속·완성 실행 수준을 정량 측정하여 실행상의 병목을 구체적으로 특정합니다.

프로그램 내 즉시 적용

실행 수준(시작·지속·완성)을 분석하여 맞춤 코칭 전략 반영. 미루기·목표 과다·시간 관리 등 원인 구체 진단.

개인 맞춤 리포트

실행 습관·시간 활용·목표 설정에 대한 평가와 보완 전략 안내. 구체적 실천 방법 중심 가이드.

프로그램 이후 활용

학습이 흔들릴 때 스스로 원인을 점검하고 조정하는 기준으로 활용. 가정에서도 환경·습관 지원 중심 개입.

진단 → 코칭 → 리포트의 연결 고리

두 진단은 단발 검사에 그치지 않고 4주 내내 코칭의 기준점으로 활용됩니다. 프로그램 종료 후에는 리포트 형태로 가정으로 연결되어, 캠프 이후에도 유지 가능한 자기주도 학습 루틴 형성의 기반이 됩니다.

  • 진단 리포트 기반 자녀 맞춤 학습 방식 적용
  • 정기적 대화로 학습 상태 점검·방향 조정

진단 결과 분석지 활용 가이드

  • 진단 결과를 바탕으로 자녀의 학습 특성·실행 패턴을 이해합니다.
  • 자녀의 유형에 맞는 학습 환경과 생활 리듬을 가정에서 조성합니다.
  • 결과를 기준으로 과정과 실행·변화를 중심으로 대화하고 지원합니다.

가정과 캠프의 동일한 언어

가정과 캠프가 같은 진단 데이터·같은 피드백 방식을 공유할 때, 학생은 일관된 메시지 속에서 습관을 형성합니다. 우리의 리포트는 학부모가 '무엇을 보고·어떻게 말할지'를 구체적으로 안내하여 가정과 캠프를 하나의 흐름으로 연결합니다.
