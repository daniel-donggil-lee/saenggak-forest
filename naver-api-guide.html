네이버 카페 API 설정 가이드

카페 자동화를 위한 1회 설정

소요시간 약 10분
언니, 카페 자동화 시스템을 연결하려면
네이버에서 "앱 등록"을 한 번만 해주시면 돼요.
아래 순서대로 따라하시면 됩니다!
0 / 8 완료
1
네이버 개발자센터 접속
1분
  1. PC 크롬에서 아래 주소를 주소창에 입력하세요
주소
developers.naver.com
  1. 오른쪽 위 [로그인] 클릭
반드시 슬기로운 초등생활 카페를 운영하는 네이버 계정으로 로그인하세요! (카페 관리자 계정)
2
애플리케이션 등록 시작
2분
  1. 상단 메뉴에서 [Application] 클릭
  2. [애플리케이션 등록] 클릭
  3. 이용약관 동의 화면 → 모두 체크 → [확인]
  4. 휴대폰 인증 요청 → 본인 번호로 인증
  5. 회사 이름에 아래 입력 후 [확인]
회사 이름
생각의숲
휴대폰 인증은 처음 1회만 하면 돼요. 다음부터는 안 물어봐요!
3
앱 이름 입력
30초

"애플리케이션 등록" 화면이 열리면:

애플리케이션 이름
원하는 이름으로 자유롭게 입력하세요
(예: 카페관리, 슬초자동화, 생각의숲 등)
이 이름은 API 관리용 라벨이에요.
카페에 표시되거나 회원들에게 보이지 않으니
본인이 알아보기 쉬운 이름이면 뭐든 OK!
4
사용 API 선택 (제일 중요!)
1분

[사용 API] 섹션을 찾아주세요:

  1. 드롭다운(선택 상자)을 클릭
  2. 목록에서 "카페"를 찾아서 선택
  3. [추가] 버튼 클릭
"카페"를 추가하면 "네이버 로그인"도 자동으로 같이 추가돼요. 이건 정상이에요!
화면에 이렇게 보여야 정상이에요:

✅ 네이버 로그인 (자동 추가됨)
✅ 카페 (내가 추가함)
5
서비스 환경 설정
2분

[로그인 오픈 API 서비스 환경] 섹션에서:

  1. 환경 선택: "PC 웹" 선택
  2. 아래 두 칸이 나타나면 정확히 입력:
서비스 URL
http://127.0.0.1
Callback URL
http://localhost:8888/callback
localhost가 아니라 127.0.0.1로 입력하세요! (서비스 URL)
Callback URL만 localhost를 씁니다.
6
등록하기 클릭
10초
  1. 화면 맨 아래 [등록하기] 버튼 클릭
  2. "등록되었습니다" 메시지가 나오면 성공!
등록 후 자동으로 "내 애플리케이션" 페이지로 이동돼요.
7
Client ID / Secret 확인
1분

"내 애플리케이션" 페이지에서:

  1. Client ID — 화면에 바로 보이는 영문+숫자 코드
    → 길게 눌러서 복사
  2. Client Secret — [보기] 버튼을 눌러야 보여요
    → 버튼 누르고 나온 코드를 복사
이런 형태로 보여요:

Client ID: A1b2C3d4E5f6G7h8
Client Secret: [보기] ← 이걸 클릭!
→ xYz9AbCdEfGh
Client Secret은 반드시 [보기]를 눌러야 나와요!
안 누르면 보이지 않습니다.
8
카페 Club ID 확인
1분
  1. 네이버 카페 접속 → 슬기로운 초등생활 카페
  2. 카페 메인에서 [관리] 클릭
  3. 관리 페이지 URL 주소창을 확인하세요
주소창이 이렇게 생겼어요:

cafe.naver.com/ManageMain.nhn?clubid=12345678

여기서 숫자 부분이 Club ID예요!
관리 페이지가 안 보이면, 카페 왼쪽 메뉴에서
"카페 관리" 또는 톱니바퀴 아이콘을 찾아보세요.

동길이한테 보내줄 3가지

1
Client ID
내 애플리케이션에서 복사한 영문+숫자 코드
2
Client Secret
[보기] 눌러서 나온 코드
3
Club ID (숫자)
카페 관리 페이지 URL의 숫자

이 3개만 카톡으로 보내주시면
나머지는 동길이가 다 세팅합니다!

궁금한 거 있으면 동길이한테 전화하세요
